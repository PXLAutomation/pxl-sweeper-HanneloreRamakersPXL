# pxl-sweeper
A tiny project scaffold for pxl-sweeper.

## Local development
Open `index.html` in a modern browser. The app uses browser-native ES modules and loads `js/game.js` from the repository root.

## Project structure
- `index.html` — static entry point
- `css/styles.css` — layout and placeholder styling
- `js/game.js` — main ES module entry point and app bootstrap
- `package.json` — module type and basic project scripts

## Verify
Run `npm test` to confirm the initial scaffold is intact.
